CMS

(Content Management System)

Qu'est ce qu'un CMS ?

Définition

Un Content Management System (en français: Système de Gestion de Contenu), C'est un outil qui permet d'administrer un site web de manière totalement graphique sans utiliser de code et sans de connaissance en programmation

Les avantages

  • Pas de licence payante pour utiliser un CMS open source

  • Simplicité de gestion et d’administration

  • Adaptation aux spécificités ou aux nouveaux besoins

  • Possibilité de travailler à plusieurs en même temps

  • Compatibilité avec les navigateurs et surtout avec les supports mobiles

Les inconvénients

  • Former les différentes personnes qui interviendront sur le site

  • Vérifier que la maintenance technique du site sera assurée

Différents types de CMS

Wordpress



Outil permettant de créer et gérer un site web

Prestashop

Outil permettant de créer une boutique en ligne
